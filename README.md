# Magic-Reader-Game-Site

Welcome to the **Magic Reader Game Site**! Build with React, Vite, and Tailwind CSS.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run dev server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Technology Stack
- **React**
- **Vite**
- **Tailwind CSS**
- **Supabase** (for backend/auth)
